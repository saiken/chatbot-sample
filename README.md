# ChatBot Sample
for study

## technology stack
### Backend
- Python 3.11.7
- Flask 3.0.0

### Frontend
- Vue.js(CDN) 3.4.5

## install
`poetry install`

## run
- pythonを起動  
`poetry run python src/app.py`

- 下記にアクセス  
`http://127.0.0.1:5000/`

## setup
### sassをcssに変換
- VSCodeにLive Sass Compiler をインストール
- sassファイルを開いて、「Watch Sass」をクリックすると自動でcssファイルが作成される
